== Description ==

Allows simple integration between your sidebar widgets and the official twitter widget 

== Installation ==

Recommend you use the built in plugin uploader which comes with WordPress, but if you prefer the manual way

1. Upload `un-official-twitter-widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Will this plugin slow my blog down =

The new twitter widget uses asynchronous javascript to load, this allows the rest of your blog to continue loading
whilst it fetches your latest tweets.
