A guide to Pointelle Slider WP Plugin

Pointelle Slider is a customizable WordPress plugin that will help you adding a very attractive featured content slider with fade or sliding effects to your WordPress site's front page, other pages, posts, sidebar widgets etc.

The posts, pages and the media library images can be easily added to the slider thru the edit panel on the WordPress admin.

The slider can be easily inserted anywhere on the site either using a template tag in the respective template file or using Pointelle Slider shortcode or using a simple widget. Also you can create multiple sliders, so that you can display different content in slider on different pages. And the best thing is, you can create multiple settings as well, and while embedding the slider you can specify which setting you would need for a particular instance of slider.

The slider can be very easily customized using the easy admin panel options. If you want to change the design of the slider with minute details, you should have some knowledge of CSS, so that you can create your own styles and change the looks of the slider as you wish.

How to install the plugin

  1. Uzip the pointelle-slider.zip folder.
  2. Upload the pointelle-slider folder to your /wp-content/plugins directory.
  3. In your WordPress dashboard, head over to the Plugins section.
  4. Activate Pointelle Slider.

Basic steps to implement the Pointelle Slider

  1. Activate the plugin.
  2. Add the posts,pages and/or media library images to the slider. You would get 'Add to Pointelle Slider' checkbox on the edit panel (of post/page/image) after plugin activation.
  3. Go to slider settings -> Change the width of the slider and the height of the slides and navigation section, depending on your theme width and your requirement. You can change other parameters like colors, borders, image size, image location and whether to show the post content or not etc.
  4. Now you can insert the Pointelle Slider on your WordPress site using either the template tag or the shortcode or the widget as follows.

Pointelle Slider Template tag

<?php if ( function_exists( 'get_pointelle_slider' ) ) {
  get_pointelle_slider(); } ?>

In case you use multiple sliders option,

<?php if ( function_exists( 'get_pointelle_slider' ) ) {
  get_pointelle_slider('2'); } ?>

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel.

Pointelle Slider Shortcode

Put the below shortcode in the post/page editor to embed the Pointelle Slider in the post/page content.

[pointelleslider]

In case you use multiple sliders option,

[pointelleslider id='2']

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel.

Pointelle Slider Widget

You have got a 'Pointelle Slider Simple' widget on Widget administration panel. Just insert the widget in any of the active widget of your theme.

Also you have got a 'Pointelle Slider - Category' widget on Widget administration panel. You can select the name of the category from the dropdown, and select which settings to apply (multiple settings can be created thru Settings admin panel).

Also you have got a 'Pointelle Slider - Recent Posts' widget on Widget administration panel. You can select which settings to apply (multiple settings can be created thru Settings admin panel) to this Slider as well.

Pointelle Slider Category Template tag

You can use the below template tag to display the posts under particular category in chronologically reverse order:-

<?php if ( function_exists( 'get_pointelle_slider_category' ) ) {
  get_pointelle_slider_category($catg_slug); } ?>

Real World Example:

<?php if ( function_exists( 'get_pointelle_slider_category' ) ) {
  get_pointelle_slider_category('social-media'); } ?>
where 'social-media' (don't forget the quotes) is the slug of the category from which you wish to pull the posts in reverse chronological order (latest is shown first).

Pointelle Slider Recent Posts Template tag

You can use the below template tag to display the recent posts added to your WordPress site/blog in chronologically reverse order:-

<?php if ( function_exists( 'get_pointelle_slider_recent' ) ) {
  get_pointelle_slider_recent(); } ?>

'Settings' Admin Panel

The Settings fall under three sections:

  1. Basic Settings Tab
  2. Left Slides Panel Tab
  3. Right Navigation Buttons Tab

Basic Settings Tab

Basic Settings

You can set the Slide transition speed, number of posts (slides) in the slider, background color, width and height of the content slides, border settings here.

Slide Transition Speed : This is the time between the transition of the slides. In case you want slow transition, enter high value like 9 or 10. If you want fast sliding, enter smaller value like 2 or 3.

Slide Transition Effect : Select the type of animation you would like while the slider slides. Pointelle Slider supports 4 different transition effects viz. Fade effect, Vertical Sliding, Horizontal Sliding and Horizontal Uncvering Transition.

Number of Posts in the Pointelle Slider : Specify how many posts you would want in Pointelle Slider, default being 3.

Complete Slider Width : If you enter 0, the slider will take the width of the container. If you enter numeric value greater than 0, it would be the width of the slider.

Slider Height : Overall height of the Slide section.

Slide Background Color : Background Color of the Slides Section.

Miscellaneous

Retain these html tags : You can retain links and br tags in the slider posts. Against this settings, put all the html tags that you wish to retain in the 'Retain these html tags' input field. Put the tags in the following manner,

<br><a><p>

Do not separate them using commas, neither use ⁄ anywhere.

Minimum User Level to add Post to the Slider : This seeting indicates which users can add or remove the posts/pages/media images to the slider. You can give the permission to editors but not authors or keep the access restricted to the 'Administrator' from here.

Randomize Slides in Slider : You can randomize the slides added to the slider, so that everytime the page on which slider is embedded loads, the slides would appear in random order to the reader.

Text to display in the JavaScript disabled browser : If the Javascript support is disabled or JS is not supported by the visitor's browser, this text would be displayed to the visitor saying that here is a slideshow which you can view in proper format if you enable the JS support on your browser.

Pointelle Slider Styles to Use on Other than Post/Pages : You can creae your own styles for Pointelle Slider. From here you can select which stylesheet should be used on general templates (i.e. like index.php or category.php or archives.php) of your site (other than single posts and pages). For the sliders embedded on posts and pages, the stylesheet specified in this settings field would be default stylesheet, but you can individually select another stylesheet from the edit post/page panel for that specific post/page.

Multiple Slider Feature : Tick this box to enable the switch to create and embed 'multiple sliders'. Once this checkbox is ticked, you would get an option to create a new slider on the 'Sliders' admin page under 'Pointelle Slider' settings. Also, you can add posts to various created sliders and embed different sliders on different pages.

Enable FOUC : In case you tick this option, while the page with the slider is loaded, you can see the Flash of Unstyled content (i.e. the content in the slider will appear without any styling while the web page is loading). This box is uncheked by default, which means the complete slider with its content will be loaded at once when the complete DOM elements (html page) is ready and loaded.

#slider_id_1 .sldr_title{background:#cccccc} 
#slider_id_2 .sldr_title{background:#ff0000;color:#fff;}

Left Slides Panel

Slide Image

These settings apply to the images extracted for the larger slides on the left hand section of the slider.

Image Pick Preferences : This setting indicates in what priority you want the images extracted for the post or the page. There are four entries, if all the checkboxes are checked that means the post/page will be scanned for the images in following sequence:

  1. Looks for an image by custom field. By default the custom field is slider_thumbnail. You can change its value, may be if you want to use the 'thumbnail' custom field.
  2. If no image is added by custom field, check for a post image/featured image set for the post (WordPresss 2.9+ feature).
  3. If no image is found, it grabs an image attached to your post.
  4. If no image is attached, it can extract an image from your post content.

Say if you uncheck the 'Consider Images attached to the post ... Order of the Image attachment to pick ' checkbox, it will skip the step 3 above.

Say, if you do not want the scanner to look to thumbnails and featured image and just directly pick the image from the post content, just untick the first three boxes and keep the fourth box checked.

Other settings include Image extract size, Display Image Size (width and height of the image) and border settings.

You can tick the check box saying 'Make pure Image Slider' in case you want only images in the slides and no content.

Post Title

These settings control the look of the title of the slide in the Left Section.

You can choose to show or do not show the Slide title, in case you only need the title in Navigation section

Slide Content

Show content in slides below title : Checking this box will enable the content/excerpt to be displayed below the title of the slide. By default this is OFF.

You can customize the looks, font, color etc. of the Slide Content area here.

You can choose how you would want the slider to pull the content i.e. 'From Post Content', 'Post Excerpt' or 'Slider Content Custom Field'. If you set it to 'Slider Content Custom Field', you should assign the value to 'slider_content' custom field as the content for the slide.

Maximum content size : You can specify the maximum content size in the slide either in number of characters or in number of words. The 'number of words' value if entered will override the 'number of characters' value. So if you want the maximum content size in 'number of characters' leave the 'numer of words' setting to blank (do not enter any value in it).

Right Navigation buttons

Navigation Button

Here you can customize the Width, Height, Background Color and Border Color of the Navigation Buttons on the right

Thumbnail Image

Here you can customize the Width, Height, and the border of the thumbnail Image in the navigation button

Title in Navigation

Here you can customize the Maximum Slide Navigation Title Size (in words) and font properties of the title in Navigation Buttons

Post Meta in Navigation

Here you can customize the content (data) pulled in this section and its font properties

Read more on Available Post Meta Functions and Parameters for Pointelle Slider here.

'Sliders' Admin Panel

Through this panel, you can remove the post/page/media image added to the slider directly or completely empty the slider. Also in case you enambe the 'Multiple Sliders Feature', you can create new sliders from this panel and delete the sliders.

Also, you can reorder the entries in the slider thru this settings panel.

FAQs

How to create Multiple Settings?

On the 'Settings' admin panel of Pointelle Slider, at the top you would see 'Create New Settings Set' button. Click the button, you would be asked to confirm that you need to create a new settings set

Once you create a new settings set, you would still be served the current settings. You can access and modify the new settings using the settings link on the right sidebar named 'Setting Panels'.

Before modifying the settings, do check on which settings panel you currently are. You can check this using the title of the Settings page which would be such as 'Pointelle Slider Settings Default', 'Pointelle Slider Settings 2' and so on

How to apply various settings to the Sliders?

Pointelle Slider Template tag

<?php if ( function_exists( 'get_pointelle_slider' ) ) {
  get_pointelle_slider($slider_id='1',$set='2'); } ?>

In case you use multiple sliders option,

<?php if ( function_exists( 'get_pointelle_slider' ) ) {
  get_pointelle_slider($slider_id='2',$set=''); } ?>

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel.

In case you enter '' (empty) for the $set, it will take the default settings.

Pointelle Slider Shortcode

Put the below shortcode in the post/page editor to embed the Pointelle Slider in the post/page content.

[pointelleslider set='2']

In case you use multiple sliders option,

[pointelleslider id='2']

[pointelleslider id='2' set='3']

Where '2' is the slider ID, which you can get from the 'Sliders' settings panel and '3' is the ID of the settings created.

How to create different stylesheets for Pointelle Sliders?

You should have FTP access to your site's folder in oredr to create your own styles.

Go to wp-content => plugins => pointelle-slider folder => css => skins

Create a new folder with a name say... 'custom'. Copy the content of 'sample' folder into 'custom' folder. Now go to custom folder => > style.css file. Open it with Notepad or Notepad++ or the editor you use to edit CSS files, add and modify/remove your own CSS rules. Save the CSS file. Upload the complete custom folder to the server, inside the skins folder.

Now on the Pointelle Slider settings page => go to 'Miscellaneous' settings => You would get value as 'custom' in dropdown for the setting 'Pointelle Slider Styles to Use on Other than Post/Pages'. Select 'csutom' and save the settings.

As well, on Post/Page edit panel, you would get an option saying 'Stylesheet to use if slider is displayed on this Post/Page ' under 'Pointelle Slider' custom meta box. Select the value as 'custom' and save the post/page.

You can create any number of style folders as 'custom' and name them whatever you want.

How to assign different navigation title than the Post/Page Title for the slide in Pointelle Slider?

By default the navigation title for a particular slider is picked as first 5 words of the post/page title.

You can change this value to custom text like 'Slide 1 Title' or 'Category A Featured Slide' or whatever using the custom field slide_nav for the post/page.

Also, you can change the 5 character restriction to 4 or 6 using the Right Navigation Button Settings => Title in Navigation => Max. Slide Navigation Title Size (in words) on Settings panel.

What are the Available Post Meta Functions and Parameters for Pointelle Slider and how to use them?

You can have atmost two Post Meta Fields namely Meta 1 and Meta 2 displayed in the navigation section of the Pointelle Slider.

The fields are pulled dynamically and you can set what meta information should be displayed in those fields.

Currently there are four Meta Functions that can be used in the settings field.

  1. pointelle_get_slide_author

    Meta Function to get author data in Post Meta field. For this function the function parameters can be WordPress 'author' object fields. For example

    • field=display_name
    • field=user_login
  2. pointelle_get_slide_pub_date

    Meta Function to get the published date of Post/Page in any format in Post Meta field. For this function the function parameters can be any of the format specified in PHP Date Manual . For example

    • format=M j, Y
    • format=Y-m-d
  3. pointelle_get_slide_category_name

    Meta Function to get the Poat category/categories in Post Meta field. For this function the function parameters can be one of the below two, the fisrt one showing the the first category to which the post belongs and the second one (all) showing all the categories to which the post belongs:-

    • show=first
    • show=all
  4. pointelle_get_slide_comments_number

    Meta Function to get the post comments number in the Post Meta field. For this function the function parameters similar to comments_number WordPress function. For example:-

    • zero=No Response&one=1 Response&more=% Responses
    • zero=No Comments&one=1 Comment&more=% comments

Also you can specify what text (with html) you would want to display before and after the meta fields.

How to add media library images to the slider?

Go to Edit Media thru Media => Library and open the image in edit mode. You would get an option to add the media image to the Pointelle Slider, you can assign the link URL to which the lside will link and you can also choose not to link a particular image to any URL.

How to to assign different 'Slider Title' for various sliders created?

In case you select multiple sliders option, you are able to create and embed different sliders with different content on various pages of your WordPress site. You may wish to have different Slider Titles for different sliders created (mostly when you are inserting the slider thru Pointelle Slider Widget). Yes, you can do that!!

When you create a new slider, you can give different names to different sliders created, which can be used as slide title. Go to Pointelle Slider 'Settings' => Slider Title settings => Pick Slider Title From => and select 'Slider Name' for the settings. Save the settings.

Also if you are using Category Widget or Reent Posts Slider Widget for Pointelle, you would get an option to enter the Slider title of your choice.

How to link the slide to different URL than the post/page permalink?

On the Edit Post/Page/Media image panel under Pointelle Slider fields, you would get an input field named 'Slide Link URL', you can input whatever URL you want the slide to link to in this field. Save the entry.

How to disable the linking of the slide to any URL?

On the Edit Post/Page/Media image panel under Pointelle Slider fields, you would get an checkbox named 'Do not link this slide to any page(url)', you can uncheck the box to choose not to link the slide to any url. Save the entry.

How to disable the promotional ads on the settings page?

Go to Pointelle Slider 'Settings' => Miscellaneous settings => Show Promotionals on Admin Page => and select 'No' for the settings. Save the settings.
