== Description ==

A simple plugin to display WordPress posts as a slideshow.

* Fullscreen slideshow of posts
* jQuery scroll animation
* 2 presentation styles included
* Randomize display order
* Custom logo options
* Description text
* Multiple resolutions

Originally designed to display real estate listings on a LCD screen.

== Installation ==

1. Upload `slidepost/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on 'Slidepost Settings' in the Settings menu
